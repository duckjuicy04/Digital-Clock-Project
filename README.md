# Mekhail Romany - Personal Portfolio

A modern, responsive personal portfolio website showcasing my skills as a **Front-end Developer**.

### Features
- Clean and minimal dark-themed design with orange accent colors  
- Animated hover effects and smooth transitions  
- Live digital clock (12-hour format with AM/PM)  
- Social media icons (GitHub, Instagram, LinkedIn)  
- "Hire Me" and "Download CV" call-to-action buttons  
- Animated button effects with ripple/offset interactions  
- Responsive stats section (Years of Experience, Projects, Happy Clients)  
- Mobile-friendly layout using flexbox  

### Tech Stack
- HTML5  
- CSS3 (with modern gradients, shadows, transitions & pseudo-elements)  
- JavaScript (real-time clock functionality)  
- Google Fonts (Nunito)  
- Font Awesome icons
